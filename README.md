# Sistema de Gestión de Películas - PCD Systems

Este proyecto es una aplicación Full Stack desarrollada como prueba técnica para la empresa PCD Systems. Permite la gestión completa (CRUD) de Directores y Películas utilizando una arquitectura desacoplada.

## Tecnologías Utilizadas

- **Base de Datos:** MySQL
- **Backend:** ASP.NET Core Web API (C#) & Entity Framework Core
- **Frontend:** Angular & Tailwind

## Estructura del Proyecto

- `/database`: Scripts de inicialización de la base de datos SQL.
- `/backend-aspnet`: API RESTful que maneja la lógica de negocio.
- `/front-angular`: Aplicación web SPA que consume la API.
